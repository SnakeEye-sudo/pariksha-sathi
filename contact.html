Reporting Issues

If you encounter any technical issues or bugs while using ParikshaSathi, please report them to us as soon as possible. Include details about:

  • What you were doing when the issue occurred
  • What browser and device you were using
  • A description of the problem or error message
  • Steps to reproduce the issue (if possible)

This information helps us fix problems faster and ensure a better experience for all users.
