## ✨ Features

- **Smart Subject Rotation** — max 2 consecutive days same subject, min 2 unique subjects/day
- **Arts-heavy priority** — History/Polity/Geography get higher frequency
- **Language/qualifying papers** scheduled last (BPSC TRE)
- **3 subjects/day** + 1 hr revision + 30 min Current Affairs
- **Mock test days** in the final 2 weeks
- **Optional subject support** for UPSC (History, Geography, Anthropology, PSIR, etc.)
- **PDF download** with Er. Sangam Krishna watermark
- **Hindi/English** language toggle
- **LocalStorage** — plan survives page refresh
- **100% free** — no login, no ads, no paywall

## 🗂️ Project Structure

```

├── index.html          # Main HTML — all screens (welcome, form, plan)

├── build.py            # Concatenates chunks/ → app.js

│   ├── c07.js          # UPSC Optional syllabuses
│   ├── c07b.js         # UPSC Optional syllabuses (extended)
│   ├── c08.js          # Core logic: getSyllabus, buildPlan, generatePlan
│   ├── c09.js          # renderPlan() — day cards, week blocks
│   ├── c10.js          # Syllabus tab renderer
│   ├── c11.js          # PYQ tips + Exam info tab
```

## 🚀 Local Development

```bash
# Clone
git clone https://github.com/SnakeEye-sudo/pariksha-sathi.git
cd pariksha-sathi

# After editing any chunk file, rebuild app.js
python build.py

# Open index.html in browser (no server needed)
```

> **Note:** Always run `python build.py` after editing any `chunks/c*.js` file. The `app.js` is the built output — don't edit it directly.
